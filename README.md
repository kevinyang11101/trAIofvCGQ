# 停车场管理系统 java501

## 项目概述

停车场管理系统是一款基于SpringBoot和Vue的前后端分离系统，具备完善的功能模块，支持用户与管理员角色，适用于多种操作系统。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- 兼容Windows和macOS操作系统

### 推荐运行环境
- jdk 1.8
- maven
- MySQL
- nodejs

### 服务端技术
- Springboot
- Mybatis-Plus

### 前端技术
- Vue
- axios
- Element-UI

## 功能模块

### 用户角色功能模块
- 个人中心
  - 查看和修改个人信息
- 登录与注册
  - 支持用户注册账号及登录
- 停车场信息浏览
  - 查看停车场相关信息
- 停车公告浏览
  - 查看停车公告
- 车位预约管理
  - 查看和管理车位预约信息

### 管理员角色功能模块
- 个人中心
  - 查看和修改个人信息
- 用户管理
  - 管理用户信息
- 停车场管理
  - 管理停车场信息
- 停车公告管理
  - 管理停车公告
- 车位预约管理
  - 管理车位预约信息
- 管理员管理
  - 管理管理员信息
